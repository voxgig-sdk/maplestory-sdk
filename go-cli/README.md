# maplestory-cli

AQL-driven CLI and REPL for the Maplestory SDK. Positional arguments are
joined into a single AQL expression and evaluated; with no arguments,
falls into an interactive REPL.

Built on `github.com/aql-lang/aql/eng/go` and the sibling Go SDK at `../go`.

## Build

```sh
go build -o maplestory-cli ./...
```

## Run

```sh
# One-shot: arguments form a single AQL expression
./maplestory-cli load 1 android
./maplestory-cli load '{id:1}' android

# REPL
./maplestory-cli
```

## Words

| Word     | Signatures                                   | Description                |
|----------|----------------------------------------------|----------------------------|
| `list`   | `[entity]` · `[query entity]`                | List records               |
| `load`   | `[entity]` · `[query entity]`                | Load a single record       |

`query` is either a Map (`{id:1}`) or a Scalar (`1`, treated as `{id:1}`).
`entity` is one of the SDK's entity names (auto-quoted as an atom).

## Entities

android avatar cache character chat cluster diff entity1 gms_new guild_mark health item job map metric mob music name npc nxf performance_metric pet quest system tip wzn wzn2 wzn3 wzn4 wzn5 wzn6 z_map

## REPL commands

- `:quit` / `:q` / `:exit` — exit the REPL
- `:help` / `:h` / `:?`     — show help
